Session Timeout Test

This page provides instructions for testing the session timeout functionality in the Flask app.

Steps to Test Session Timeout:

    Enter a valid username and password (user1/password1):



  1. After logging in, you will be redirected to the Dashboard.
  2. Refresh the Dashboard page or navigate to another page within the app.
  3. Look at the network call to the /dashboard endpoint. Note that in the Cookie header value there is a session token. Get the curl and run it in a terminal. You full access to this users account now without having to know the password.

Note: The session timeout is set to 5000 days for demonstration purposes. In a real-world scenario, session timeout values will be much shorter.
